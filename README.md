# 🔐 Security Incident Reporting System

## Project Overview
This system allows cybersecurity professionals to record security incidents, display incident reports, and search for incidents by Incident ID. Designed specifically for Tanzania-based organizations to track and manage security threats effectively. The system includes user authentication to ensure only authorized personnel can access incident data.

## Degree Program
**Cyber Security and Digital Forensics Engineering**
## Group Number
**11**
## Features
- ✅ User registration and login (User Management Module)
- ✅ Record security incidents with unique ID
- ✅ Display all incident records in a table
- ✅ Search incidents by Incident ID
- ✅ Incident status tracking (Under Investigation, Critical, Resolved)
